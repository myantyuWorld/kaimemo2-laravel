# 画面一覧

家計簿アプリケーションの画面一覧。要件定義書に基づいて各画面の概要を定義する。

## 画面一覧表

| 画面ID | 画面名 | 処理概要 |
|--------|--------|----------|
| SCR001 | メイン画面（ダッシュボード） | カレンダー形式での月間支出表示、今日の支出一覧、各カテゴリの予算残額、買い物メモへのアクセスを表示する |
| SCR002 | 支出入力画面 | 支出データ（日付、品目、金額、カテゴリ）の入力フォームとカテゴリ選択機能を提供する |
| SCR003 | カレンダー画面 | 月間カレンダーで各日の支出総額を表示し、日付タップで詳細な支出内訳を表示する |
| SCR004 | 支出一覧画面 | 指定した日の支出一覧（品目、金額、カテゴリ）をスクロール不要で表示する |
| SCR005 | 予算管理画面 | カテゴリ別予算設定と使用状況の可視化を行う |
| SCR006 | 買い物メモ画面 | 買い物リストの表示・編集および共有メンバーの管理を行う |
| SCR007 | 集計画面 | 期間指定による支出集計とグラフ・チャートによる可視化を提供する |
| SCR008 | ログイン画面 | ユーザー認証（ログイン・ログアウト）機能を提供する |
| SCR009 | ユーザー管理画面 | 家族メンバーのユーザー登録・管理と権限管理を行う |

## 画面詳細

### SCR001: メイン画面（ダッシュボード）
- **関連機能**: F001-2, F001-3, F002-2, F003
- **アクセス**: ログイン後の初期画面
- **特記事項**: アプリの中心となるハブ画面。カレンダー形式での月間支出表示がメインコンテンツ

### SCR002: 支出入力画面
- **関連機能**: F001-1
- **アクセス**: メイン画面から遷移
- **特記事項**: 1日の終わりのまとめ入力を想定。日付と、システムに登録されているカテゴリ分の金額入力フォームがある。

### SCR003: カレンダー画面
- **関連機能**: F001-3
- **アクセス**: メイン画面で表示（デフォルト表示）
- **特記事項**: 月間カレンダーで各日の支出総額を表示。日付タップで詳細表示

### SCR004: 支出一覧画面
- **関連機能**: F001-2
- **アクセス**: カレンダー画面で日付タップ時に表示
- **特記事項**: 一覧性を重視したレイアウト。カテゴリ別支出内訳表示

### SCR005: 予算管理画面
- **関連機能**: F002-1, F002-2
- **アクセス**: メイン画面から遷移
- **特記事項**: 視覚的な予算使用状況表示。食費・日用品・お小遣いカテゴリの管理

### SCR006: 買い物メモ画面
- **関連機能**: F003-1, F003-2
- **アクセス**: メイン画面から遷移
- **特記事項**: 複数ユーザー間での共有機能

### SCR007: 集計画面
- **関連機能**: F004-2
- **アクセス**: メイン画面から遷移
- **特記事項**: 要約情報中心のデータ表示

### SCR008: ログイン画面
- **関連機能**: F005-1
- **アクセス**: アプリ起動時
- **特記事項**: プライベート利用のアクセス制御

### SCR009: ユーザー管理画面
- **関連機能**: F005-2
- **アクセス**: 管理者権限でアクセス
- **特記事項**: 家族メンバー登録・権限管理
